LLM-агент для работы с командной строкой. Плюс может программировать, гуглить, короче, делать всё, что можно сделать через bash. Теоретически очень универсальный агент, на практике - сильно зависит от интеллектуальности вашей LLM.
Может рассуждать.

Библиотеку chat bot вы не найдёте - предполагается, что у вас есть какой-то способ запускать LLM. Интерфейсы такие:
Класс должен называться ChatBot, при инициализации должна быть входная переменная user_id
И нужен метод
.create_text(prompt, max_tokens, temp, stop_words, verbose)
